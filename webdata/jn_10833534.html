日本将允自卫队海外救侨民时动用无后座火箭炮_军事频道_中华网 | | | | | | | | | | | | | | | | | | 当前位置： > > > 正文 日本将允自卫队海外救侨民时动用无后座火箭炮 2013-10-14 09:26:57 【 】 　　 　　日本自卫队使用的无后座火箭炮。(资料图) 　　日本新闻网13日报道，日本政府计划修订自卫队法，允许自卫队去海外救助侨民时，可以携带反战车用的小型重武器无后座火箭炮。 　　迄今为止，日本内阁会议决定的自卫队员前往海外救助侨民时，只允许携带手枪、步枪和机关枪，而且运输工具也只允许船舶和飞机。 　　日本政府认为，去年发生在非洲的日本公司职员遭绑架和袭击事件，给政府如何救助在海外的侨民提出了一个很大的课题。从事件发生地到机尝码头如何安全运送侨民，显然需要考虑配备陆上运输工具，同时为了保证陆上运输的安全，应该配备重武器。 　　消息称，日本政府正在制定中的自卫队法修正案，将提出允许自卫队在海外救助侨民时，使用装甲车等安全运输工具，同时允许携带无后座火箭炮，以防备恐怖组织车辆等的追击，阻止自爆恐怖活动的发生。 (责任编辑：CM013)相关报道: 分享到： 已有人参与 条评论 文明上网，请登录后发表评论。 所有评论仅代表网友意见。 更多 的相关消息 相关报道 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 日本侨民,日本,日本自卫队,日本内阁,无后座火箭炮,海外行动,海外救助 　　 　　 　　 | | | | | | | | 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 网上不良信息举报电话：(010)84105871 　　 | | | | | 　　版权所有 中华网<script type="text/javascript">ac_as_id = 2384;ac_click_track_url = "";ac_format = 0;ac_mode = 1;ac_width = 280;ac_height = 210;</script><script type="text/javascript" src="http://static.acs86.com/g.js"></script><img src="http://dvsend.china.com/cc/00KXFhttp://dvs.china.com/385/dot.gif" width="0" height="0" border="0">
