河北武邑涉事民警被调离、直接领导被免职 　　 　　 　　 　　网友纷纷晒“衙门”门难进、脸难看、事难办 潘石屹：跑11趟公安局没办成转户口 　　 　　 　　 　　“门难进、脸难看、事难办”，是官僚主义作风的典型表现，老百姓最反感，因为它真折腾人、真耽误事。河北省公安厅12日晚间通报，针对央视报道的该省武邑县公安局出入境工作人员态度粗暴、刁难办证群众的问题，涉事民警和主管领导分别被给予党纪、政纪处分。 　　 　　 　　 　　央视曝光 　　 　　 　　 　　大半年返乡6次才办下护照 　　 　　 　　 　　11日，央视《焦点访谈》报道，户口在河北武邑的小周，目前在北京工作。去年底公司要派他出国，需要办因私护照，由于在北京缴纳社保不足一年，按规定他必须回户口所在地办理。大半年过去了，小周返乡5次，护照一直没有办下来，每次都被告知需要补充不同的材料，且办事人员态度蛮横。 　　 　　 　　 　　不久前，记者和小周一起来到了武邑县公安局出入境科。出入境科的办公室面对面坐着两名办事人员，两人都没穿警服，其中一名一直看着报纸，头始终没抬一下。看报纸的女士就是前几次接待小周的办事人员，她报纸看得很专注，直到对面的同事叫她，她才如梦初醒般抬起了头，似乎对打扰自己读报有些不满意。 　　 　　 　　 　　这已经是小周第五次来办护照了，前几次他都是无功而返，原因是材料不齐。而这次他自认为让准备的材料都备齐了，应该没有问题了，结果这名办事人员又发现了新问题。原来小周的身份证是在内蒙古读大学时办的，不是本地的，他要在这里办护照，必须提供本地的身份证才可以。为了身份证小周又要折腾一次了。第六次，小周的护照终于办了下来。 　　 　　 　　 　　报道中，当事人小周最后感慨地说，“权力岂能玩弄百姓。”这话引起了不少人的强烈共鸣。 　　 　　 　　 　　公安厅厅长 　　 　　 　　 　　谁找群众麻烦，就让谁有麻烦 　　 　　 　　 　　报道细数了小周补办的证明：1.无犯罪证明；2.公司在职证明；3.公司营业执照；4.公司外派资格证明；5.本地身份证。而实际上，像小周这样的普通公民办理因私护照，其实只需要提供身份证和户口簿及复印件，然后就是照相填表就行了。上述那些办事人员让他补办的证明，除本地身份证需补办外，其他的依法都不需要。 　　 　　 　　 　　11日晚央视报道播出后，河北省公安厅立即派出由纪委、督察、出入境等部门人员组成的联合调查组，赴武邑县连夜开展调查。12日上午，河北省公安厅厅长董生说，坚决避免再次发生类似问题，对违反警规禁令的事件，要发现一起、严肃处理一起，绝不姑息迁就。对不适合在公安机关工作的民警，要及时清理出去，“谁给人民群众找麻烦，我们就一定让他有麻烦１ 　　 　　 　　 　　据了解，根据干部管理权限，衡水武邑县纪委、监察局已经对两名涉事民警、主管领导分别予以党纪、政纪处分。武邑县公安局出入境管理大队民警史某已被调离公安机关，留党察看两年，行政降两级；免去武邑县公安局出入境管理大队大队长房某职务，给予党内严重警告，行政记大过处分；给予分管领导王某党内严重警告处分。 　　 　　 　　 　　 分享到 0 | | | | | | 2013/10/14 6:59:28
