日本另辟蹊径争钓鱼岛话语权 和中国韩国对抗_军事频道_中华网 | | | | | | | | | | | | | | | | | | 当前位置： > > > 正文 日本另辟蹊径争钓鱼岛话语权 和中国韩国对抗 2013-10-14 09:37:33 【 】 　　 　　资料图：钓鱼岛 　　【环球时报驻日本、韩国特约记者 文玉 王刚】据《日本经济新闻》13日报道，为了加强在争议领土问题上和中国、韩国对抗，日本将在近期推出对外宣传网站，以英语、汉语、法语等11国语言宣扬日本对这些领土的主权主张。评论称，这是对中韩在国际舞台积极推销本国主张的回应。报道称，日本曾多次通过外务省网站主张对独岛(日本称竹岛)和钓鱼岛的主权，但实际效果不大，因此改弦更张另辟蹊径。 　　《日本经济新闻》称，日本国内外宣传领土与主权的相关专家恳谈会曾于今年7月汇总的一份建议书认为，围绕领土摩擦问题，有必要向当事国之外的第三国介绍日本的立场和主张。于是，日本政府便开始着手制作宣传日本立场的动画。该报道称，作为该行动的第一波，日本将于近日在该网站上公开一段长30秒的动画。今后除了继续制作关于钓鱼岛和独岛的动画之外，计划在明年3月前制作包括北方领土在内的共计12条动画。据悉，这些动画将陆续在即将开设的对外宣传的专用网站上公开，还将用于驻外使领馆的宣传活动中。此前，日本关于领土的宣传内容多被放在外务省网站上，日本政府相关人士认为这些内容被其他海量的信息所淹没，效果不佳。该日本政府相关人士期待通过专用网站发布动画信息，加强他国对这些问题的关心和理解。 　　此外，随着靖国神社秋季大祭的临近，日本首相安倍晋三是否会参拜也成为日本舆论关注的话题。日本《产经新闻》12日称，安倍正在该问题上施展暧昧战术，不说去也不说不去。12日受访时，被问及是否会在本月17日至20日的秋季大祭中参拜靖国神社时，安倍表示：十分遗憾的是，(参拜靖国)已成为外交问题的情况下，去还是不去，我无法回答。他还说：我想对为国捐躯的英灵们表达崇敬之意、感谢之意、哀悼祈福。12日，当被问及同样的问题时，日本内阁官房长官菅义伟也拒绝发表评论。《韩国日报》13日评论称，安倍本人应该认识到参拜靖国神社绝不应该当外交牌来使用。最近美国国务卿和国防部长访问日本都没有到访靖国神社，而是祭奠了离此不远的千鸟渊战殁者公墓，这实际上是亲自给日本政治人物作出示范。 (责任编辑：CM013)相关报道: 分享到： 已有人参与 条评论 文明上网，请登录后发表评论。 所有评论仅代表网友意见。 更多 的相关消息 相关报道 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 日本,钓鱼岛,舆论战,宣传,西方国家,安倍晋三 　　 　　 　　 | | | | | | | | 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 网上不良信息举报电话：(010)84105871 　　 | | | | | 　　版权所有 中华网<script type="text/javascript">ac_as_id = 2384;ac_click_track_url = "";ac_format = 0;ac_mode = 1;ac_width = 280;ac_height = 210;</script><script type="text/javascript" src="http://static.acs86.com/g.js"></script><img src="http://dvsend.china.com/cc/00KXFhttp://dvs.china.com/385/dot.gif" width="0" height="0" border="0">
