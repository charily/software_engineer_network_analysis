美军遗骸迎回仪式被爆造假 老兵被愚弄极愤怒_军事_环球网 ||| 您的位置：>>>正文 美军遗骸迎回仪式被爆造假 老兵被愚弄极愤怒 2013-10-14 08:10 字号：TT 　　 　　遗骸到达仪式曾被视作对美军阵亡者的无比尊重。 　　【环球时报特约记者 张亦驰】装载美军阵亡将士遗骨的棺椁由大型运输机运送到美军基地，棺椁上覆盖着美国国旗，仪仗队将阵亡将士抬下运输机……这种庄重的仪式极尽哀荣，让人感到美国对阵亡军人的高度尊重。然而，据英国《每日电讯报》网站12日报道，多年来美国国防部一直在阵亡将士遗骸回国仪式上造假。报道称，仪式中形如棺材的箱子中装的遗骸早已经回国几个月而非刚刚抵达，而仪式上的运输机也往往是临时从机库拉出来充当道具的。 　　遗骸抵达仪式被指造假 　　《每日电讯报》网站这篇题为五角大楼在战死军人遗骸抵达仪式上造假的文章称，五角大楼承认，在美国军方为越南战争、朝鲜战争和第二次世界大战期间失踪人员遗骸举行的极具爱国主义情怀的抵达仪式中，其遗骸并非当天刚刚返回的。几年来，五角大楼一直为失踪人员遗骸举行感人但是虚假的抵达仪式。举行仪式的时候，眼泪汪汪的家庭成员和老兵们天真地认为运输机中覆盖着国旗、看上去像棺材的箱子中就是当天荣归故里的美军牺牲人员遗骨。而如今，五角大楼承认，这个仪式中，其实并没有刚刚从国外战场运回的死难者遗骨，而仪式中的运输机也只是个装饰品，有些甚至根本不能飞，只是在仪式举办前拖到现常 　　报道称，在组织者当中，仪式造假是公开的秘密。在覆盖着星条旗的棺木中存放的遗骸通常已经返回美国几个月了。在举办仪式那天，它们被从对其进行鉴定和分析的实验室拉出来，参加完仪式后再送回实验室。 　　造假的仪式演了七年 　　据美国NBC新闻网报道，这种抵达仪式已经进行了7年，通常1年举办4次，在仪式上公众可以看到C-17运输机，从其尾部的舱门中卸下覆盖着国旗、形状如棺材一样的容器，在观众唱完国歌后，由仪仗兵抬到停机坪，然后一名海军陆战队士兵吹响军号。到场者包括军方高官、民间的权贵和受到邀请的老兵以及失踪人员家庭成员。他们相信那架飞机刚刚载着他们的亲人或战友返回美国。 　　报道称，该仪式的具体组织者就是战俘及战争失踪人员联合调查司令部(JPAC)。战俘及战争失踪人员联合调查司令部位于夏威夷，主要负责在越南战争、朝鲜战争和第二次世界大战期间失踪的大约8.3万名失踪军人遗骸寻找、复原和鉴定工作。这个旨在负责搜寻并还原美军失踪将士遗骨的机构成立于1973年，它的前身位于泰国，3年后迁至美国夏威夷的西卡姆空军基地，现共有425名工作人员。目前，该机构每年大约鉴定80具遗骸，这里的实验室处理每具遗骸平均需要11年，耗资超过100万美元。 　　被指用死人欺骗活人 　　五角大楼解释称，抵达典礼应该被描述为荣誉典礼，并称将修改名称。一位JPAC的发言人说，‘抵达仪式’对人造成了误导。此外，五角大楼坚持称看上去像棺材的容器内确实是最近从古老战区运回的遗骸，只不过不是举办仪式那天返回的。五角大楼强调，其人员极度仔细、认真、诚实和尊重地处理这些遗海但五角大楼的声明没有解释，既然没有遗骸刚刚抵达为什么这个仪式被称为抵达仪式，或者为什么公众被告知那天早晨从实验室运过来的遗骸将要被运往这个实验室开始鉴定程序。 　　《每日电讯报》称，参加仪式的宾客都说，仪式给他们留下清晰的印象，那就是遗骸刚刚抵达。曾参加过二战和朝鲜战争的一位美空军退伍老兵称，他已经参加过超过50次这种仪式，他深信飞机刚刚在几小时前抵达。他对自己被愚弄感到极为愤怒。有网友指出，美国军方组织这种仪式无非是为了向活人显示其对阵亡将士的尊重，但是这种把死人当道具搬来搬去的仪式，难道不是对那些死者的不恭，对生者的不敬吗？ 免责声明版权作品，未经《环球时报》书面授权，严禁转载，违者将被追究法律责任。 　　责任编辑：周扬 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-10-14 07:54:00 2013-10-12 08:28:00 2013-10-12 08:20:00 2013-10-12 08:20:00 2013-10-12 08:15:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
