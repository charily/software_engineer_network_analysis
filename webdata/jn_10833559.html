我军大规模演习作战方案漏洞令参谋长直冒汗_军事_环球网 ||| 您的位置：>>>正文 我军大规模演习作战方案漏洞令参谋长直冒汗 2013-10-14 09:02 字号：TT 　　 　　【】 　　战役筹划：中军帐充满火药味 　　使命行动-2013B跨区机动战役演习见闻 　　一梯队的登陆行动，没有充分考虑到潮汐对航渡的影响，建议重新部署！×地域村庄密集、地域狭小，不适合开辟机降区域，建议进行调整！ 　　今天，使命行动-2013B跨区机动战役演习进入战役筹划阶段。记者探访战役集群作战会现场，迎面扑来浓烈的火药味，陆、海、空三军参演部队指挥员围绕战役行动展开激烈讨论。交战尚未开始，中军帐内早已硝烟弥漫。 　　此前，战役集群参谋长林火茂根据最新战场态势，提出决心调整建议后，三军指挥员随即走到沙盘前，综合战场信息提出意见建议 ‘敌’在A地域配属了近40%的装甲力量，我摩步营在A地域机降，岂不成了‘瓮中之鳖’？6类重点打击目标针对性都很强，但忽视了B海域的海底光缆，没能从根本上毁伤其通信枢纽，破袭其体系。 　　这时，陆航作战群、特种作战群等各群指挥员也通过视频系统，源源不断地传来意见建议。一条条作战建议，直奔主题、直击要害，直指作战方案的疏忽和漏洞。林参谋长感慨地对记者说：有的意见建议让我们直冒汗，但字字都实在，句句都在理！ 　　把所有的问题解决在‘战斗’打响之前，作战会就应该这样开！战役集群指挥长刘小午介绍，为提高战役筹划质量，他们坚持贯彻群众路线，充分发扬军事民主，广泛听取各军兵种指挥员意见建议，让作战决心方案最大程度与实战对表。 刘指挥长综合各方意见建议，及时调整了兵力部署等。几分钟后，他走到沙盘前，对4个作战方向、6个作战编组和10余个作战群的兵力配置、作战时间和协同要点等，迅速定下作战决心。 　　记者注意到，整个作战会议和作战协同，所有指挥员没有一个人照着稿子念，全是对照沙盘摆兵布阵。 按打仗的标准改进训风演风，必须首先从‘中军帐’改起。演习总导演、广州军区政委魏亮告诉记者，他们把反四风的成果落实到演习中，坚决抵制演习编脚本、念稿子、走过场现象，切实把作战会开成诸葛亮会，把作战协同开成揭露问题会，真正让战役筹划细致、精准、务实，锤炼参演部队谋划指挥打仗的能力。 　　战斗虽然还未打响，但透过火热的中军帐，记者感受到，一场场充满硝烟味的联合大战，已然在求真务实的运筹帷幄中酝酿发酵…… （李华敏石斌欣记者严德勇） 　　责任编辑：刘昆 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-10-08 09:01:00 2013-09-28 10:06:00 2013-09-25 15:04:00 2013-09-21 10:02:00 2013-09-18 09:25:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
