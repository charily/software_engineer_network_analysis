日外交密电证实政府曾刻意回避东南亚慰安妇问题_军事频道_中华网 | | | | | | | | | | | | | | | | | | 当前位置： > > > 正文 日外交密电证实政府曾刻意回避东南亚慰安妇问题 2013-10-14 08:26:49 郭一娜 【 】 　　新华网东京10月13日电（记者郭一娜）日本《朝日新闻》13日披露，日本政府在上世纪90年代曾刻意回避调查东南亚慰安妇问题，以减少国际社会对这一问题的关注。 　　《朝日新闻》当天在头版头条报道了这一新闻。报道援引一份1993年7月30日的日本外交密电说，时任日本外相武藤嘉文指示日本驻印度尼西亚、马来西亚、菲律宾三国使馆说，调查会煽动各国民众情绪，所以要尽量避免在三国的调查。 　　报道指出，尽管日本政府摆出对慰安妇问题进行全面调查的姿态，但实际上并未这样做。时任日本内阁高官曾表示，对于慰安妇问题的调查不仅限于朝鲜半岛。 　　针对日本政府未就慰安妇问题进行充分调查一事，印尼政府曾于1992年发表声明予以谴责。 　　1993年，时任日本内阁官房长官河野洋平就慰安妇问题调查结果发表谈话，承认日军直接参与在朝鲜半岛、中国等地设置慰安所及强征当地妇女充当慰安妇，并对此表示道歉和反剩这段讲话被称为河野谈话。 　　舆论普遍认为，日本政府在慰安妇问题上的态度，是衡量其历史认识的一面镜子。去年，日本现任首相安倍晋三曾表示有意修改河野谈话，引发亚洲国家以及国际社会对安倍政权否定历史的警惕。 (责任编辑：CM013)相关报道: 分享到： 已有人参与 条评论 文明上网，请登录后发表评论。 所有评论仅代表网友意见。 更多 的相关消息 相关报道 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 慰安妇,侵略历史,日本,东南亚,密电,日本外交,慰安所,河野谈话 　　 　　 　　 | | | | | | | | 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 网上不良信息举报电话：(010)84105871 　　 | | | | | 　　版权所有 中华网<!--SMARTmedia Begin:--><SCRIPT type="text/javascript" SRC="http://smjs.allyes.com/sm.js"></SCRIPT><SCRIPT type="text/javascript" SRC="http://smjs.allyes.com/china++inpage++sm++show.js"></SCRIPT><!--SMARTmedia End--><img src="http://dvsend.china.com/cc/00KJGhttp://dvs.china.com/385/dot.gif" width="0" height="0" border="0"><script type="text/javascript">ac_as_id = 2384;ac_click_track_url = "";ac_format = 0;ac_mode = 1;ac_width = 280;ac_height = 210;</script><script type="text/javascript" src="http://static.acs86.com/g.js"></script><img src="http://dvsend.china.com/cc/00KXFhttp://dvs.china.com/385/dot.gif" width="0" height="0" border="0">
