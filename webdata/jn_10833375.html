小s被轰退出主持 电视台：相信会撑过去_娱乐_环球网 ||| 您的位置：>>>正文 小s被轰退出主持 电视台：相信会撑过去 2013-10-14 07:47 字号：TT 　　 　　小S（资料图） 　　据台湾媒体报道，小S因涉胖达人代言不实与炒股两案遭检调传讯，台湾名嘴周玉蔻日前在TVBS《2100全民开讲》狠批小S不配当公众人物，应离开主持岗位，小S目前只主持中天《康熙来了》，中天13日则表示相信小S会撑过去。 　　小S主持《康熙来了》10年，从偶像到3个女儿的妈，中天认为她敬业，将最好、最有效果的节目内容呈现给观众，胖达人事件爆发，她8月26日向社会大众鞠躬，并说：我是公众人物，既然推荐了，就有义务向社会道歉。《康熙》一直是台湾综艺节目的指标，她主持功力无庸置疑。 　　生病中的高凌风昨也以调查中不该定罪、观众爱看小S，周玉蔻你凭什么、你跟她什么关系、藉小S提升知名度控诉周。周说：娱乐圈同仁水准如果这么低，我就不多说了，大家看新闻就知道他们在炒股，许父子把面包当成工具。 　　周批小S后，她的微博遭大批网友围剿，落井下石还不是时候，管好你自己先、人家一有什么事就急著往下踩，见不得人好，她先前与小S为黑人潮T事件对呛，有网友质疑她记仇，最可恨的是和别人结下梁子后，百般为难人家，周说：网友评论，不值得回应。 免责声明版权作品，未经环球网Huanqiu.com书面授权，严禁转载，违者将被追究法律责任。 　　责任编辑：西综 分享到： 请选择您浏览此新闻时的心情 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
