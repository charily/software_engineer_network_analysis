美媒：中国军方到访因美国政府“关门”取消_军事_环球网 ||| 您的位置：>>>正文 美媒：中国军方到访因美国政府“关门”取消 2013-10-14 08:36 字号：TT 　　 　　资料图：姚云竹少将 　　【环球网综合报道】据美国《外交政策》双月刊网站10月11日报道，联邦资金的削减正在妨碍美国在全球的外交努力。一支中国将军代表团访美的计划已被取消。美国国务院已被迫推迟了预定在日内瓦举行的美国人权记录评审会议。高级外交、贸易和军事会议已被搁置。 　　文章称，中国一些最具影响力的军事思想家和政策制定者(包括数位将军)本来预定近日赴美，在美国陆军军事学院出席一系列筹划已久的会议，随后到华盛顿一些更加著名的智库进行私人讨论。 　　报道指出，这支代表团由中国人民解放军军事科学院的姚云竹少将带领。在北京与华盛顿因从网络安全到南中国海等议题而对峙之际，这支代表团在美国的会议被认为很重要。 　　但是，美国陆军10月9日表示，陆军方面不得不取消这些会议，因为已没有资金接待中国代表团。陆军在一封电子邮件中指出：等美国民主程序给陆军提供资金以进行国际活动后，我们期待在双方都方便的最早可能时间重新安排此次交流活动。 　　报道称，美国国会不能给政府提供资金还导致了更多戏剧性事件：幼儿园关闭、战斗机飞行员停飞、电码译员玩弄着拇指、无人检查石油管道、伤心的家庭不能看到死在战场上的儿子。与这些人间惨剧相比，美国的外交失误似乎无关紧要。但是，这些失误确实能改变世界对美国的看法，真希望是微小的改变。 　　布鲁金斯学会约翰·桑顿中国中心主任乔纳森·波拉克说：当你出现此类状况、当你取消访问计划时，尤其是提前通知对方的时间非常非常短时，这会导致一个仍然谨慎看待美国意图的国家对美国的信任度降低。 免责声明版权作品，未经环球网Huanqiu.com书面授权，严禁转载，违者将被追究法律责任。 　　责任编辑：刘昆 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-10-09 09:23:00 2013-10-03 08:06:00 2013-09-29 10:04:00 2013-09-27 14:15:00 2013-09-26 08:25:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
