巴勒斯坦官员否认阿拉法特死于中毒的说法_国际新闻_环球网 ||| 您的位置：>>>正文 巴勒斯坦官员否认阿拉法特死于中毒的说法 2013-10-14 05:17 字号：TT 　　巴勒斯坦官员13日否认英国医学期刊关于巴前领导人阿拉法特死于中毒的说法，称巴方目前尚未接到正式尸检结果。 　　阿拉法特死因调查委员会主席陶菲克·提拉维当天告诉新华社记者，正式尸检结果目前处于保密阶段，还没有递交给巴方，巴方也没有得到递交结果的确切时间。 　　提拉维说，英国研究报告并没有包含任何新内容，所作出的关于阿拉法特死于中毒的推断是基于卡塔尔电视台去年的一个调查性报告。 　　英国知名医学期刊《柳叶刀》网站12日刊登最新研究报告称，研究人员在阿拉法特体内及衣物中发现大量放射性物质同位素钋210，由此支持阿拉法特中毒身亡的说法。 　　正式尸检结果原定于9月递交巴方。据消息人士称，目前阿拉法特遗孀苏哈正在和负责尸检的瑞士放射物理学研究所协商，决定是由苏哈还是巴勒斯坦民族权力机构接受尸检结果。 　　2004年10月，阿拉法特由拉姆安拉前往法国贝尔西军医院接受治疗。同年11月11日，阿拉法特在法国逝世。去年7月，媒体披露阿拉法特生前衣物被发现含有高剂量放射性物质钋，由此引发阿拉法特可能是中毒身亡的猜测。巴方随后同意为阿拉法特开棺验尸以查明其真正死因。去年11月，瑞士、法国和俄罗斯专家从位于巴勒斯坦拉姆安拉的阿拉法特墓地提取其尸骨样本，由调查组专家带回各自国家化验。(记者陆佳飞 吕迎旭) 　　责任编辑：赵衍龙 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-08-14 05:41:00 2012-12-03 10:17:00 2012-11-30 11:28:00 2012-11-29 13:02:00 2012-11-28 09:13:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
