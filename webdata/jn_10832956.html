英国宣布将简化赴英签证手续以吸引中国游客_国际新闻_环球网 ||| 您的位置：>>>正文 英国宣布将简化赴英签证手续以吸引中国游客 2013-10-14 07:37 字号：TT 　　【环球网报道记者周旭】据新加坡《联合早报》10月14日消息，正在中国访问的英国财政大臣奥斯本宣布，英国政府将简化中国游客签证手续。报道指出，英国希望能够借助此举大幅提升中国游客数量。 　　奥斯本说，改革现有签证手续规定对英国商业和旅游业来说都是好消息。据悉，简化签证手续后，前来欧盟国家旅游的中国人将不再需单独申请英国签证。获授权的中国旅行社将可以通过提供申根签证申请表，一站式申办申根和英国签证。 　　10月12日，奥斯本抵达北京开始对中国为期五天的访问，同时，伦敦市长约翰逊也开始其中国行程，两人分别带领着一些英国商界代表。报道称，奥斯本14日将在北京大学发表演讲，并与约翰逊一起回答学生提问。奥斯本还将参加中英经济金融对话。之后他将前往深圳、广州参观访问，随后前往香港。 免责声明版权作品，未经环球网Huanqiu.com书面授权，严禁转载，违者将被追究法律责任。 　　责任编辑：方芳 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-10-08 02:35:00 2013-09-30 02:35:00 2013-09-23 09:43:00 2013-09-18 07:47:00 2013-09-10 14:50:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
