斯诺登称美监控项目让美国人更不安全 伤害国家_军事频道_中华网 | | | | | | | | | | | | | | | | | | 当前位置： > > > 正文 斯诺登称美监控项目让美国人更不安全 伤害国家 2013-10-14 09:35:24 【 】 　　据外电13日报道，美国中情局前雇员斯诺登日前说，美国在全世界侵入电话与互联网系统大规模的监控项目让美国人更不安全。 　　在星期五(11日)维基解密网站公布的简短录相中，斯诺登说，美国的监控项目让我们面临与自己政府冲突的危险。 　　他说，美国的监控技术将所有人置于监控之下，即使在不需要的时候也在监控。 　　他说，这伤害了美国的经济，伤害了国家。他们限制了人们说话、思考、生活、创造、自由交际与交朋友的能力。 　　这是7月12日，斯诺登在莫斯科机场露面，对俄罗斯当局提出避难申请以来首次公开发声。俄罗斯当局8月1日决定给予斯诺登庇护。 　　斯诺登说，美国政府不愿意起诉那些在国会撒谎，在摄像机前对国家撒谎的高级官员，但他们在迫害那些勇于说出真相的人却是肆无忌惮。 　　在随同录像公布的消息中，维基解密网站说，斯诺登是在星期三(9日)在莫斯科接受萨姆亚当斯奖时发表上述讲话的。为斯诺登颁奖的4名美国前政府官员分别为前FBI官员劳里、前国安局官员德雷克、前司法部官员拉达克和前中情局官员麦戈文。 (责任编辑：CM013)相关报道: 分享到： 已有人参与 条评论 文明上网，请登录后发表评论。 所有评论仅代表网友意见。 更多 的相关消息 相关报道 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 斯诺登,监控技术,网络监控,中情局,避难,NSA,国家安全局,美国政府,美国人 　　 　　 　　 | | | | | | | | 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 　　 网上不良信息举报电话：(010)84105871 　　 | | | | | 　　版权所有 中华网<script type="text/javascript">ac_as_id = 2384;ac_click_track_url = "";ac_format = 0;ac_mode = 1;ac_width = 280;ac_height = 210;</script><script type="text/javascript" src="http://static.acs86.com/g.js"></script><img src="http://dvsend.china.com/cc/00KXFhttp://dvs.china.com/385/dot.gif" width="0" height="0" border="0">
