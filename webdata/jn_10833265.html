13:07:42 来源： 光明日报 　　 　　 　　 　　1月3日,俄罗斯总统梅德韦杰夫正式签署了《优化俄联邦国家机关公务员数量》的命令,该命令指出,俄联邦政府今后将在3年内削减公务员的20%,并严格控制政府机构人员的增长。 　　 　　 　　 　　2010年6月8日,梅德韦杰夫责成俄政府研究将全国公务员数量削减20%的可能性。当时俄舆论指出,俄总统提出大幅削减公务员的建议,旨在提高政府效率、减少财政开支以及提升政权机关在民众中的威望。2010年8月,俄财政部遵照梅德韦杰夫的指示,拟定了一份为期3年的规划草案,核心内容是“裁减官员数量,缩减财政开支”。 　　 　　 　　 　　俄罗斯历届政府都曾尝试精简机构和裁员,但官员却越裁越多,政府机关“臃肿”依旧。据俄国家统计局提供的数据,截至2010年6月,包括俄联邦立法、执行和司法机关在内的俄国家公务员数量已超过了60万人。 　　 　　 　　 　　按照俄财政部的规划,俄在2011年4月1日前要将全国公务员数量裁减5%,2012年4月1日前将公务员数量裁减10%,2013年4月1日前将公务员总量最终裁减20%。也就是说,如今在任的政府公务员中,有1/5的人,即120507名官员将下岗,财政预算可节约430亿卢布开支(1美元约合30卢布)。裁员节省下来的收入,一半将用于提高公务员的工资,另一半将贴补它用。 　　 　　 　　 　　几乎对所有国家的政府来说,公务员人数的削减都是一个难题。一方面,政府的运作有赖于公务员的维持;另一方面,在整体社会架构中,公务员又是一个既得利益集团,减员往往遭遇来自国家机关的强大阻力。梅德韦杰夫也承认,“这一措施毫无疑问是极为残酷的。” 　　 　　 　　 　　俄此次大幅裁减公务员,其鲜明的政策取向在于,减少财政开支、提高政府效率,提升行政机关在民众中的威望。因为日益膨胀的政府机构、庞大的公务员队伍已成为政府开支的沉重包袱,甚至成为提高政府效率的绊脚石。 　　 　　 　　 　　近年来,因为“工作好、薪水高、前途光明”,公务员已成为俄年轻人择业的首眩全俄社会调查中心的最新民调结果显示,超过50%的18-30岁的俄罗斯人希望自己成为公务员。 　　 　　 　　 　　国际金融危机以来,俄罗斯到处都在裁减人员、下调工资,但政府公务员的工资水平并未下滑。据最新统计,俄罗斯居民目前月平均工作为两万卢布,而莫斯科市公务员的月平均工资是4.21万卢布,莫斯科州的公务员是4.46万卢布,圣彼得堡市的公务员是4.96万卢布。 　　 　　 　　 　　梅德韦杰夫的这一总统令还将送交俄联邦会议(议会)、联邦审计署等部门审议。这些部门在审议之后将就裁员问题向联邦政府提出合理化建议。 　　 　　 　　 　　梅德韦杰夫同时指出,该计划不应机械地进行,应按照实际情况循序渐进,其中靠国家财政支持的医生、教师等行业人员不在裁员之列。 　　 　　 　　 　　对于俄政府此次裁员计划,虽然俄社会各界仍有怀疑声音出现,但是大部分人对政府此举表示了欢迎。 　　 　　 　　 　　俄国家战略委员会主席霍米亚科夫表示,对总统来说,这将是一项十分艰难的任务。但这一问题必须解决,因为这有利于政府工作效率的提高。 　　 　　 　　 　　有不少人认为,俄罗斯公务员在人口总数中所占比例,并不比英国等发达国家高。俄民众之所以认为有必要削减公务员队伍,主要因为俄公务员队伍“官僚、腐败、低效”。 　　 　　 　　 　　但更多俄人士则认为,此次裁员依然是“说起来容易,做起来难”。一旦裁减付诸实施,肯定会出现诸多意想不到的问题和矛盾。 　　 　　 　　 　　按：一会宣布全民免费医疗，一会裁公务员这是怎么啦？ 分享到 0 | | | 本版块主题总数：2188401 / 帖子总数：71023680 　　 今日论坛共发帖：7870 / 昨日发帖：41012 / 最高日发帖：75754跳转论坛至：
