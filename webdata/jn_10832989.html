　　来自中新社（2013-10-13）的消息说，中国政府力推为每个少数民族拍一部电影，因为“至今还有20多个少数民族没有本民族题材的电影”。 　　 　　看了这个消息，我立马想起了一个俚语：狗拿耗子。 　　 　　拍电影这样的事情，应该是文艺家的本职工作。文艺家们为什么不去拍民族题材的电影，从文艺这个“行当”的角度来看，可能是没有这方面的灵感。文艺创作是需要灵感的，没有灵感别说拍电影了，恐怕写封家书都难。那么，文艺家的创作灵感从何而来呢？ 　　 　　来自生活。 　　 　　电影是一种艺术美，但是这种艺术美蕴藏在平凡的生活当中，就像金子“躲”在矿石里面一样。发现这样的艺术美，需要一双“慧眼”，能在一般人视为寻常的地方看出非凡来。比如说吧，面对着一口不足半亩的小小池塘，能够写出“半亩方塘一鉴开，天光云影共徘徊，问渠那得清如许，为有源头活水来。”这样的诗句，一般人能行么？ 　　 　　“为有源头活水来”，这是文艺家们创作灵感唯一的来源所在。生活就是他们的“源头活水”。 　　 　　长期以来，中国文艺家们的“源头活水”变成了“死水微澜”。什么文艺要为无产阶级政治服务啦，什么文艺要歌颂D啦，歌颂SHZY啦，等等等等，不一而足。实际上，所有这一切都是套在文艺家们头上的紧箍咒，极大地限制了文艺家们的创作空间。孙悟空在花果山的时候，多么自由自在，大闹天宫都不在话下。可是，等到他被唐僧收编了，尤其是那个老妖婆观音菩萨诓他戴上了紧箍咒之后，连打真正的妖魔鬼怪都困难重重。失去了自由的人，思想僵化，创作灵感自然也“魂飞魄散”了。60多年来，中国出了什么值得人们珍惜的文艺作品？啊，有个中国作家得了一个国家大奖。很是了不起。但是，读者需要的可能还不是什么国际大奖，而是像《红楼梦》那样的不朽作品。永垂不朽的是后者，而不是前者。我相信，中国在相当长的一个时期之内，能够拿得出手的，依然还是“老祖宗”，不是“后起之秀”。 　　 　　这倒不是中国的“后起之秀”们无能，而是他们的才能被一张无形的大网罩住了，就像孙悟空被那座“五指山”压住了一样，大闹天宫的英雄好汉也变成了囚徒，还能怎样？ 　　 　　所以，解放“囚徒”，这是包括电影艺术在内的中国文艺繁荣昌盛的不二法门。解放“囚徒”的关键，就是给文艺家们自由，让他们天马行空，任意驰骋，独来独往。所有的艺术都是“个人主义”的结果。我在南昌参观八大山人的作品，看到那些别具一格的画作，真是觉得不可思议。画家真的是“个人主义”到家了，随心所欲，我行我素。画家好像一点儿也不遵循什么“艺术创作规律”，想怎么画就怎么画。多亏了这样的“个人主义”，才给我们留下了如此宝贵的精神财富！如果当年也有人要求他为什么什么服务，中国还有八大山人么？ 　　 　　今天的中国，同样存在着不少的“八大山人”，可惜的是，他们没有朱耷那样好的“命运”。不管怎么说，朱耷画画是自由自在的。现实当中的“朱耷”们可就不行了，“笼子”多得很，动辄得咎，他们只能生活在“五指山”下。 　　 　　或许，正是考虑到了这样一点，我们的人民政府才“赤膊上阵”要去拍电影。我当然相信，政府直接出面是能够拍出电影来的，尤其是有关少数民族题材的电影。不过呢，这样拍出来的电影是否一定能够反映今天中国少数民族的真实面貌，那就是另外一个问题了。今天的中国，民族问题好像并非“到处莺歌燕舞”。政府直接出面拍出来的电影敢于反映那些非“莺歌燕舞”的一面么？《五朵金花》是反映大跃进的，真实的大跃进在这部电影里面一个字也没有说。这个历史事实可能不是电影家们造成的吧。 　　 　　孔子说“君君臣臣父父子子”，有人说孔子在这里搞等级制。当然不能排除这种可能。不过，我们是否也可以从“各守其位，各尽其职”这样积极的方面来看待这句话呢。政府有政府的职能，文艺家有文艺家的职能，各司其责岂非理所当然，何必越俎代庖？政府应该去干自己职能范围之内的事情，像拍电影这样的事情，最好让那些行家里手去干。中国的文艺家们，只要有了真正的创作自由，什么样的电影都可以拍出来，而且一定能够青出于蓝而胜于蓝。 　　 　　现在的问题是，我们的人民政府敢不敢给文艺家们真正的创作自由。 　　 　　 　　 　　 分享到 0 　　 　　 　　 　　 延伸阅读|最新热帖 [思项羽] [netbug2008] [没有螃蟹] [yungui] [横竖有理] [一家村主] [人民的巨掌] [秋秋秋A] [周杨1920] [行道者] [大灰猫] [剁辣椒] | | | 本版块主题总数：2188339 /
