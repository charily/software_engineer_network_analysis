奥巴马否决提高政府债务上限计划 谈判再陷僵局_国际新闻_环球网 ||| 您的位置：>>>正文 奥巴马否决提高政府债务上限计划 谈判再陷僵局 2013-10-13 10:17 字号：TT 　　原标题：关门12天 美两党谈判再度陷入僵局 　　国际在线报道（记者 吕晓红）：美国政府部分关门已经过去12天了，距离美债违约大限也只剩5天了，处在这场政治飓风中心的众议院共和党人12日宣布，奥巴马总统否决了他们提出的提高政府债务上限的临时短期计划，将谈判的舞台让给了参院共和党人。同一天，参院民主党提出的提高债务上限议案因未能获得法定的60票支持而流产。 　　12日一大早，众院共和党人开了一个简短的内部会议，之后宣布奥巴马总统否决了他们的计划，他们也再提不出什么新的计划，只好看参议院那边的结果了。这意味着众院共和党人决定将这场围绕着预算与债务上限的驴象两党之争中共和党一方的前线交给了参院共和党人。 　　随后，参议院照议程对民主党周三提出的提高政府债务上限提案举行程序性投票表决，结果是53对45，两党阵营泾渭分明，这个结果未能达到法定的60票支持票，因此议案流产。对此，来自纽约州的民主党参议员舒曼表示非常担忧。这是在玩火。我们不知道什么时候市场会对此作出反应，因为谁也不敢确保在下周四之前不出现动荡。我非常担心周一美国股市开盘之后，或许就因为今天的投票而造成投资者的恐慌，继而引发股市下跌，借贷利率上涨，美国政府债券贬值。这个问题非常严重。 　　而缅因州共和党参议员柯林斯的提案也遭到了否决。该提案建议提高政府债务上限至明年1月底；并按当前的（也就是全面减支后）的财政标准（每年9670亿美元）来维持联邦政府的运转，期限至明年3月；同时要求废除奥巴马医改法案中规定的医疗设备税。 　　眼下，众议院大部分议员已经离开华盛顿回家休假了，因为下周一是美国的公共假日哥伦布日，纪念发现美洲的航海家哥伦布。参院明天还会开会，但不会举行任何投票表决。因此，美国政府财政预算、债务违约危机谈判可以说是再度陷入僵局。 　　当地时间10日晚奥巴马总统在白宫会见众院共和党领袖之后，双方对外放风说会谈是有益的；11日上午奥巴马总统又在白宫同参议院共和党人会面，双方继续放风说是建设性的，据美国媒体披露，这也是相持不下的双方为稳定市场而达成的唯一的协议，就是对外保持一致的口径。 　　参院多数党领袖里德12日也表示，当天上午他同参院少数党领袖、共和党议员麦康奈尔终于有了一次面对面、认真的对话。我们的对话非常真诚，当然还处在初级阶段，没有任何结果。我希望我们的讨论对于美国人民来说是一种安慰吧。 　　与此同时，奥巴马总统呼吁双方尽快就提高政府债务上限达成协议，他在每周例行讲话中强调说，双方的共识是：应避免美国不能履行承诺而带来的经济后果。对美国主权信誉评级的损害不仅会令全球市场失控，也将使每个美国人的借款变得更加昂贵，由此美国的每个家庭和企业都要承担他所说的‘共和党违约税’。 　　奥巴马总统与民主党的立场十分坚定，那就是先恢复联邦政府的全面正常运行，再来谈相关的改革。而共和党人则要求先谈判再依照法律程序恢复正常。可以说，现在的局面跟整个僵局发生之初、甚至之前的情况没有什么根本性的变化。 　　责任编辑：刘洋 分享到： 请选择您浏览此新闻时的心情 相关新闻 2013-10-11 15:35:00 2013-10-11 13:24:00 2013-10-10 10:54:00 2013-10-09 09:05:00 2013-10-08 08:03:00 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
