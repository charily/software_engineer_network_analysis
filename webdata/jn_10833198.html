曝足协3官员竞聘被劝退因特殊规定 张剑私下道歉_体育_环球网 ||| 您的位置：>>>正文 曝足协3官员竞聘被劝退因特殊规定 张剑私下道歉 2013-10-14 08:23 字号：TT 　　两个足球博士、一个海归足球硕士，且在中国足协一线工作时间都在10年以上的三个人，却无法按照正常组织程序竞聘体育总局组织的足协部室正职。这不是笑话，是发生在刚刚结束的足协竞聘中的真事。 　　足协竞聘报名前，这三位高学历的人员被足协领导暗示，不要报名参加竞赛部和技术部这两部门正职竞聘，因为这两部门正职人选已被提前确定。为了完成组织程序，竞聘演说要进行，因此要求三位高学历官员甚至连竞聘演说都不要参与了。 　　而《新京报》在报道中也表明，此次戚军与杨新利的竞聘成功，也是因为有一条规定：总局直接派遣的干部，可以直接任命。足管中心目前中层干部中只有技术部主任杨新利和竞赛部主任戚军是总局直接派遣的干部，两人在竞聘报名时报的是自己现在任职部门，所以才会出现我们上面所说的情况。 　　在足协10个多月的履历中，足协掌门人张剑极度低调，却不得不在官场潜规则面前低头。他在中层正职竞聘结束后，私下和遭受不公平竞聘遭遇的官员道歉，并暗示自己也是受到更高层压力，才不得不按照领导要求行事，希望下属能够理解自己的处境。 　　尽管下属对张剑此番道歉表示理解，同时也都情愿或不情愿退出竞赛部和技术部竞聘，但同时也和张剑表示，这两部门是足协核心部门，总由外行人管理，难免闹出各种笑话和纰漏。一旦出事，承担管理责任的还是张剑。张剑对此也表示无奈，服从上级命令是他首要选择。至于出了事谁担责任，则是后话。 　　责任编辑：麻霏 分享到： 请选择您浏览此新闻时的心情 图片推荐 2345678910 关联媒体 　　 环球时报客户端 　　 　　 推荐服务 | | | | | | | | | | ? 环球网
